# Hotel reservation backend

## Project outline
- users -> book room from a hotel
- admins -> going to check reservation/bookings
- authentication and authorization -> JWT tokens
- Hotel -> CRUD API -> JSON
- Rooms -> CRUD API -> JSON
- Scripts -> database management -> seeding, migration
